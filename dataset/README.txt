This project uses the Lending Club 2007-2018 dataset.  For some reason the availablity online is a little shakey and the initial source is no longer available. A mirror of the data exists at https://www.kaggle.com/wordsforthewise/lending-club, to download, run, from within this folder: 'kaggle datasets download wordsforthewise/lending-club', then extract and rename 'accepted_2007_to_2018Q4.csv' to 'loan.csv'
